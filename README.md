# 社区爱心养老管理系统 python404

## 项目概述

社区爱心养老管理系统是基于Python语言开发的一套便捷、高效的管理系统。它通过信息化手段，为社区养老提供全面的管理与服务平台，旨在提升社区养老服务的质量和管理效率。

## 技术栈

- **后端：**
  - Python 3.7
  - Django
  - MySQL 5.7
- **前端：**
  - Vue.js

## 功能模块

### 用户功能

- **首页浏览：** 用户可以通过首页导航条进入各功能页面。
- **个人中心：** 允许用户更新个人信息，并进行以下详细操作：
  - 个人信息管理
  - 身体健康信息管理
  - 借阅信息查询
  - 归还信息查询
  - 活动信息浏览与报名
  - 地址管理
  - 收藏管理

### 管理员功能

管理员具备以下操作权限：

- 首页内容管理
- 用户管理
- 身体健康信息管理
- 公共书籍管理
- 借阅信息管理
- 归还信息管理
- 还书入库操作
- 图书分类管理
- 活动信息管理
- 活动报名管理
- 活动分类管理
- 系统设置管理
- 管理员个人信息管理

## 系统角色

- **管理员：** 负责系统的日常管理和维护工作。
- **用户：** 在系统中注册并使用各项养老服务。

## 运行环境

- 后端：需安装Python 3.7环境，配置好Django框架及MySQL数据库。
- 前端：基于Vue.js构建，在现代浏览器上运行。

## 目录结构

```plaintext
社区爱心养老管理系统/
├── backend/                # 后端目录
│   ├── apps/               # Django应用目录
│   ├── db/                 # 数据库相关
│   ├── static/             # 静态文件目录
│   └── ...
└── frontend/               # 前端目录
    ├── src/                # Vue源码目录
    ├── public/             # 公共文件目录
    └── ...
```

## 核心亮点

- **前后端分离：** 前端使用Vue.js，后端使用Django，利于项目维护与扩展。
- **用户友好：** 界面清晰，操作简便，便于老年人使用。
- **管理高效：** 管理员功能全面，有效提升管理效率和服务质量。
